## What "App Store listing" actually is

Not a separate account to create — it's a feature inside **App Store Connect**, which you already have access to (the "Welcome to App Store Connect" email arrived when your Apple Developer Program enrollment activated). When Phase 10 arrives, listing the app means: creating an app record there (using the `com.tapowner.app` bundle ID already reserved via EAS), then filling in — name, subtitle, description, keywords, screenshots (taken from the real running app), a **support URL** and **contact email** (this is where row 9's real inbox becomes necessary — Apple requires a working contact, not a placeholder), a privacy policy URL (build doc has this as a separate Phase 10 to-do), pricing (free, no in-app purchases in v1), category, and age rating. Then you submit a build from EAS for Apple's review. None of this needs to happen now — it's explicitly Phase 10 — but now you know what it involves.
